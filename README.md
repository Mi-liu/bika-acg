# pic-age

哔咔漫画的 PC 版本，基于 Vue 3、Vite 7、TypeScript、Pinia 和 Element Plus 开发。
当前仓库已配置 Cloudflare Workers 静态资源部署，部署说明见 [CLOUDFLARE.md](./CLOUDFLARE.md)。

## 使用说明

- 图片默认开启隐私模式，会以灰色遮挡展示。可在「用户头像 - 设置 - 漫画阅读」中关闭「隐私模式」。
- R18 内容默认关闭。可在「用户头像 - 设置 - 漫画过滤」中开启「R18 内容」，开启后会显示排行榜、热搜、随机漫画和相关分类入口。
- 主题默认跟随系统。可通过顶部栏主题按钮，或在「用户头像 - 设置 - 外观主题」中切换为浅色模式、深色模式或跟随系统。
- 设置页支持导出、导入和清空本地数据，包含阅读设置、外观设置、稍后再看、关注作者和分类缓存等本地数据。

## 本地开发

```sh
pnpm install
pnpm dev
```

常用检查命令：

```sh
pnpm type-check
pnpm lint
pnpm build
```

## 对比哔咔网页版，有什么区别

- 漫画列表、详情和阅读页适配 PC 端浏览体验。
- 支持关注作者、稍后再看，进入详情后会自动从稍后再看中移除。
- 支持自动阅读，可配置自动滚动速度。
- 支持隐私模式、图片质量、阅读宽度、API 代理线路和文件代理线路设置。
- 支持 R18 内容开关，关闭时隐藏或限制相关入口。
- 支持浅色、深色和跟随系统主题。
- 支持分类和作者过滤，并可管理本地数据备份。
